# YAT — Yet Another Typeless

[繁體中文](./README.md) | [English](./README.en.md)

YAT 是一個桌面語音輸入工具。按住快捷鍵開始說話，YAT 會把語音轉成文字；如果你已設定文字潤飾，還會進一步去掉贅詞和重複、整理段落、補上標點、修正術語，最後直接貼到目前的應用程式，或保留在剪貼簿裡。

## 為什麼會想用 YAT

- 在任何 App 裡都能用同一個熱鍵開始輸入
- 說完就能直接貼上，不用來回切換視窗
- 中英混說也能保留原本語感，不會硬被統一成單一語言
- 需要時再加上 AI 潤飾，讓結果更接近可直接送出的文字
- 如果自動貼上不適合你的工作流，也可以只寫入剪貼簿

## 適合誰

- 想得比打字快的人
- 想用口述寫訊息、文件、筆記或 commit message 的開發者
- 經常中英夾雜、需要保留原語感的人
- 想減少鍵盤負擔、改用更輕鬆輸入方式的人

## 開始使用

1. 從 [Releases](../../releases) 下載並安裝 YAT
2. 開啟 YAT，從系統匣圖示進入設定
3. 到「**語音辨識**」分頁選擇 **Groq** / **OpenAI**，或填入自訂 OpenAI 相容端點
4. 貼上 API Key，按「**測試連線**」
5. 想讓文字更乾淨，再到「**文字潤飾**」分頁設定 LLM
6. 直接按住快捷鍵開始說話

> **小提醒**
> - 只要先設定好語音辨識，就可以開始使用；如果尚未設定文字潤飾，YAT 會直接輸出轉錄結果。
> - 預設熱鍵是 **按住說話**：Windows / Linux 為 **右 Ctrl**，macOS 為 **右 Cmd**。
> - 預設輸出模式是 **自動貼上**，同時會把結果保留在剪貼簿。

## 使用流程

1. 按住快捷鍵開始錄音
2. 浮動狀態窗（Capsule）會顯示錄音狀態、經過時間與麥克風音量
3. 放開快捷鍵結束錄音；錄音或處理中可按 **Esc** 取消
4. YAT 會把語音送去做轉錄
5. 若已啟用文字潤飾，YAT 會再整理標點、段落、條列與術語
6. 結果會直接貼到目前 App，或寫進剪貼簿

## 你會喜歡的地方

### 更像輸入工具，而不是單純逐字稿

- **全域熱鍵**：支援按住說話、單一按鍵、連按兩下、組合鍵
- **自動貼上 / 僅複製**：可以積極，也可以保守
- **浮動狀態窗（Capsule）**：錄音時不必切回主視窗，也能知道現在進行到哪一步
- **系統匣常駐**：平常安靜待命，需要時再叫它出手

### 更像能直接送出的文字

- **AI 潤飾**：去掉贅詞、重複與口吃，補標點、分段、整理條列
- **保留中英混說**：不硬把語碼轉換改成單一語言
- **詞彙表**：搭配文字潤飾時，修正品牌名、產品名和專有術語更穩
- **自訂指令**：用幾句簡單規則就能微調輸出風格；有需要再進階改 prompt

### 為日常工作流準備好的細節

- **麥克風選擇**：多裝置環境更好管理
- **音效提示**：不用盯著畫面，也知道何時開始、結束或出錯
- **錄音時自動靜音**：Windows / macOS 可在錄音期間暫時靜音系統音訊
- **歷史紀錄**：可搜尋、複製、刪除與重試先前結果
- **介面偏好**：支援深色 / 淺色 / 跟隨系統，並可切換繁體中文與英文
- **開機自動啟動**：讓 YAT 一登入就待命

## 設定分頁

| 分頁 | 你會在這裡做什麼 |
|---|---|
| **總覽** | 檢查語音辨識 / 文字潤飾是否就緒，快速看目前狀態與最近輸出 |
| **一般設定** | 調整熱鍵、輸出模式、麥克風、主題、介面語言、音效、自動靜音、逾時、重試與開機自啟 |
| **語音辨識** | 設定 STT 服務商、Base URL、API Key、模型與語言提示 |
| **文字潤飾** | 啟用或關閉 LLM，並設定服務商、Base URL、API Key 與模型 |
| **提示詞** | 加上簡單輸出規則，必要時再編輯進階 system prompt |
| **詞彙表** | 新增常見術語與專有名詞修正 |
| **歷史紀錄** | 搜尋、複製、重試結果，並清理過期紀錄 |

## API Key 與推薦組合

YAT 採用 BYOK（Bring Your Own Key）：你使用自己的 API Key，請求會直接送到你設定的服務商。

| 服務 | 註冊 / 控制台 | 適合拿來做什麼 | 常用模型 |
|---|---|---|---|
| **Groq** | [console.groq.com](https://console.groq.com) | 入門最快、速度好、免費額度友善 | STT：`whisper-large-v3-turbo` / LLM：`llama-3.3-70b-versatile` |
| **OpenAI** | [platform.openai.com](https://platform.openai.com) | 想用官方 OpenAI 相容 API | STT：`whisper-1` / LLM：`gpt-4o-mini` |

> **建議入門組合：**
> 先用 **Groq STT** 把語音輸入跑順，再視需求決定是否打開文字潤飾。

## 平台提醒

- **macOS**：自動貼上與全域熱鍵通常需要「輔助使用」權限；也別忘了開麥克風權限。
- **Windows**：如果目標 App 是用系統管理員權限執行，自動貼上可能失敗；這時改成「僅複製」通常最穩。
- **Linux**：全域熱鍵目前需要 X11 或相容層；不同桌面環境對模擬貼上支援差異很大，若不穩，建議改用剪貼簿模式。Linux 版本目前也不支援錄音時自動靜音。
- **取消鍵提醒**：在 Linux 上，按 **Esc** 取消時，該按鍵仍可能同時送到前景 App。

## 隱私與資料

- **API Key 只存本機**：會儲存在你的裝置上，並只送到你設定的端點
- **音訊不落地**：錄音只在處理期間存在記憶體中，不會被長期儲存
- **歷史紀錄在本機**：轉錄結果存放於本機 SQLite 資料庫，預設保留 30 天，可在設定中調整
- **沒有遙測**：YAT 不會回傳使用統計
